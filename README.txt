# WeatherStation_Server

This folder contains all the server scripts.
In order to achive the functinality there is a need to create the database using mysql_db.sql commands and connect to the database.
In order to do it login details such database, user, password needs to be changed in connect.php.
For beeing able to display data using eg.index.php (make sure that the table is not empty as empty table will not display data).


Suporting script:
upload.sh - used to upload script to the server 
download.sh - used to download scripts from the server on my local PC

index.php
output.php

forecast.php
